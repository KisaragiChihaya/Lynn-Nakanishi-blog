Cloudflare Workers 建站体验

前言

各位可能未曾注意到，由我运营的如月云其实已经上线很久了，只不过以前一直使用 Freenom.com 的免费域名运营，当时的架构是 VPS + zfile（服务端） + Nginx（本地反向代理），这套架构刚开始没有什么问题，但是可能是因为我使用的 VPS 性能不够（因为没钱租用更好的）所以 zfile 总是被系统当作无用进程杀死，效果不甚理想。近日因为安全问题 加上我忘记按时续期导致现在过期后要付费续期 ，我重新注册了现在的域名并且重构了网站架构，重构过程中我顺带体验了一下 Cloudflare 提供的 Cloudflare Workers 功能，发现十分好用，于是决定分享一下使用心得。
