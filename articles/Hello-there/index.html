Hello there.

终于还是重新开设了博客。

还记得最早开始对博客感兴趣大概是五、六年前。当年除了兴趣以外什么都不会，凭着一腔热血以及对新事物的好奇，我在国内某套路云平台以超低的价格购入了人生中第一台虚拟主机（然后发现自己被坑了），并且使用了一键脚本部署了 LNMP+Wordpress。本以为自己会断断续续更新文章，奈何当时学业繁重加上新政策出台，这个博客便因域名无法备案而关闭了。

而后过了几年，技术也没有什么长进，唯一学会的只有如何正确的使用搜索引擎。通过搜索引擎我也找到了很多优秀的资源，于是便萌生了重新开博客写作的想法。鉴于之前糟糕的备案体验，我选择了一台地理位置位于东京的 VPS，并选择了大家常用的宝塔面板重新建立了新的博客。网站架构依然是熟悉的 LNMP+Wordpress，不过我尝试安装了一些优化性能和改善管理体验的插件（Wordpress 的臃肿程度众所周知）。新博客稳定运行了差不多一年，结果因为忘记续费被服务商注销了服务器，而我又没有开启自动备份（因为要加钱，而我很穷 www），为此我对开博客这件事产生了一点心理阴影……从此好久不碰博客。

再次决定开博客其实也是几个月之前的事，因为我开始尝试做偶像研究了。新企划总得有个官网吧？自己写？实力不允许；Wordpress？体验不好；Typecho？没用过不熟悉……经过各种对比之后我选择了 Hexo ，一个优美的静态博客系统，并将 GitHub 作为代码管理平台，利用 GitHub Pages 做出了企划的官网。体验到了 Hexo 的便捷性之后我便又想开始折腾，于是利用闲暇时间找到了现在这个漂亮的主题 AD ，再配合 Valine 和 Leancloud 一起实现评论功能，一个全新的博客就诞生了。事实证明，它完全满足我的需要，这样就足够了。

至于博客的名字“中岛”，是取自于“中西的岛”，毕竟这也算是我的避风港了。

希望这次能好好坚持下去吧。

最后，让我用一个简短的小程序结尾吧。
