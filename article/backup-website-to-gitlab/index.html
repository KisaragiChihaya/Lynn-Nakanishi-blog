将网站备份到了 GitLab

好久没更新了，稍微写一下这个网站的近况。

因为本人最近访问 GitHub 常常被重置连接，所以为了保险起见，本人将博客备份到了一份到 GitLab。测试了一下整体速度还算可以，所以就又传了其他几个仓库上去，未来也应该会一直使用 GitLab 做备份了。当然本人的 GitHub 也会继续存放一些代码或文档，并会尽量做到两边同步更新，如果有意愿参与翻译可以在任意一个平台提交推送请求，随时欢迎。

それじゃあ。
